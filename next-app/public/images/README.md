# Required images for ohmyglass.ca

Place these image files in this folder so they are served at `/images/...` and load on the live site (including Vercel). Files in `static/` are copied to the build output as-is.

## Required files

| File | Used for |
|------|----------|
| `hero.jpg` | Homepage hero background (fallback) |
| `hero.webp` | Homepage hero background (preferred) |
| `logo.png` | Header and footer logo (fallback) |
| `logo.webp` | Header and footer logo (preferred) |
| `window-repairs.jpg` | Services card (fallback) |
| `window-repairs.webp` | Services card (preferred) |
| `aluminum-storefront.jpg` | Services card (fallback) |
| `aluminum-storefront.webp` | Services card (preferred) |
| `door-repairs.jpg` | Services card (fallback) |
| `door-repairs.webp` | Services card (preferred) |
| `custom-mirror.jpg` | Services card (fallback) |
| `custom-mirror.webp` | Services card (preferred) |
| `og-image.jpg` | Open Graph / social sharing (1200×630 recommended) |
| `favicon.ico` | Browser tab icon |

After adding these files, commit and push so Vercel deploys them and images load on the app.
